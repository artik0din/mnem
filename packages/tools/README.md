# @mnem/tools

Tool format adapters for exposing a Mnem vault to LLM agents.

Provides `toOpenAITools(vault, options)` (OpenAI function-calling format) and `toAnthropicTools(vault, options)` (Anthropic tool-use format). Operations and path scope can be restricted per-agent.

Implementation scheduled for Mnem v0.1.
